# `eslint-config-rusoul-react`



## Usage

### Install
```
npm i -D eslint eslint-config-rusoul-react
```

### Configuration
```
// eslintrc.js
module.exports = {
  extends: ['eslint-config-rusoul-react'],
  globals: {},
  rules: {},
}

```

## Requirements
```
eslint 8+
```
